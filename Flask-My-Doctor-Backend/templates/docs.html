Chatbot API Documentation

POST /api/chat-bot

This API takes a JSON message as input and returns a response.

{
  "message": "who is 7up"
}
        
{
  "response": "7 Up is a brand of lemon-lime-flavored non-caffeinated soft drink. 
  The rights to the brand are held by Keurig Dr Pepper in the United States and 
  by 7 Up International in the rest of the world."
}

application/json

  • 400 Bad Request: If the request does not contain a valid "message" field.
